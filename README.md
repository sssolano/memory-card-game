# Balatro Memory Card Game 🃏
 
A memory card game built with vanilla HTML, CSS, and JavaScript, inspired by the aesthetic of [Balatro](https://www.playbalatro.com/).
 
## How to Play
 
- Flip two cards at a time to find matching pairs
- You have 8 lives — each failed match costs one
- Find all 7 pairs before running out of lives to win
## Built With
 
- HTML, CSS, JavaScript — no frameworks, no libraries
